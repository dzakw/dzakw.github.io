HTML Formatting

  • Inline styles have the highest specificity, overriding any other styles defined in internal or external stylesheets.
  • <i> tag is used when you want to italicize the text.
  • <b> tag is used when you want to bold the text.
  • <strong> tag is used when aside from making the text bold, you also want to emphasize it.
  • <em> tag is used when you aside from making the text italic, you also want to emphasize it.
  • <del> tag is used to delete the text, as if it was there but then it was removed, while the <s> tag is used to just strike the text.
  • <ins> tag is used to insert the text, as if it was not there but then it was added, while the <u> tag is used to just underline the text.
  • <mark> tag is used to highlight the text.
  • <q> tag is used to make a short quote (usually browser will use the quotation mark ").
  • <blockquote> tag is used to quote paragraph from other source.
    usually shown inside of a blockquote (which will be indented)
  • <abbr> tag is used to make the text abbr.
  • &lt; is used to display the less than (<) sign.
  • &gt; is used to display the greater than (>) sign.
  • &amp; is used to display the ampersand (&) sign.
  • &nbsp; is used to display a non-breaking space (like when you type people's name that contain space, it wont break into new line).
  • This is a small text.
  • However, the <big> tag is strongly discouraged to be used in HTML5, use CSS styling instead.
  • <sub> tag is used to make the text subscript.
  • <sup> tag is used to make the text superscript.
  • <address> tag is used to define the contact information of the author/owner of a document or an article like this:
    Written by John Doe.
    Visit us at:
    Example.com
    Box 564, Disneyland
    USA
  • <cite> tag is used to define the title of a work, usually written in italic
    Example: Harry Potter and the Deathly Hallows by J.K. Rowling
  • <bdo> tag is used to override the current text direction, the dir attribute can be filled with rtl for right to left, or ltr for left to right
    Example: This text will be written from right to left

HTML Links

  • <a> tag is used to create a hyperlink, the href attribute is used to specify the URL of the page the link goes to.
  • Like in this example: Visit our HTML tutorial
  • Notes: the target attribute is used to specify where to open the link,
  • The value _blank will open the link in a new tab.
  • The value _self will open the link in the same frame as it was clicked (default behavior).
  • Other value of target (like _parent or _top) can be used to open the link in different frame or window, which are quite obsolete in the modern websites
  • Images and icon can also be used as a link, like this:

    • the syntax are as follows:
                        
    <a href="https:github.com/dzakw" target="_blank"> <i class="fa-brands fa-github" style="color: #28433a; font-size:100px"></i></a>
  • links can be absolute URLs and relative URLs
  • Absolute URLs are URLs that include the full URL of the page, like this is link

    •                     
    <a href="https://github.com/dzakw">this is link</a>
  • Relative URLs are URLs that only include the path to the page, like this is link
    the syntax are as follows:

Section 2: CSS

CSS Syntax

  • As mentioned before, CSS can be used in 3 ways: inline, internal, and external.
    The syntax of CSS is as follows:
    •                 
    selector {property1: value1; property2: value2; propertyN: valueN;}
  • to link to external CSS file, use the <link> tag in the <head> section of the HTML file, like this:
                    
    <link rel="stylesheet" href="styles.css">
