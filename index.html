HTML Formatting

Inline styles have the highest specificity, overriding any other styles defined in internal or external stylesheets. <i> tag is used when you want to italicize the text.
<b> tag is used when you want to bold the text.
<strong> tag is used when aside from making the text bold, you also want to emphasize it.
<em> tag is used when you aside from making the text italic, you also want to emphasize it.
<del> tag is used to delete the text, as if it was there but then it was removed, while the <s> tag is used to just strike the text.
<ins> tag is used to insert the text, as if it was not there but then it was added, while the <u> tag is used to just underline the text.
<mark> tag is used to highlight the text.
<q> tag is used to make a short quote (usually browser will use the quotation mark ").
<blockquote> tag is used to quote paragraph from other source.

usually shown inside of a blockquote (which will be indented)
<abbr> tag is used to make the text abbr.
&lt; is used to display the less than (<) sign.
&gt; is used to display the greater than (>) sign.
&amp; is used to display the ampersand (&) sign.
&nbsp; is used to display a non-breaking space (like when you type people's name that contain space, it wont break into new line).
This is a small text.
however, the <big> tag is strongly discouraged to be used in HTML5, use css styling instead.
<sub> tag is used to make the text subscript.
<sup> tag is used to make the text superscript.
